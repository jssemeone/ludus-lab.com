Этот симулятор генерирует криптографический ключ на основе энтропии движения курсора.

Каждые 2 секунды фиксируется позиция курсора. Из неё вычисляется бит по формуле:
БИТ = (X + Y + ВРЕМЯ) % 2

После сбора 16 бит формируется мастер-ключ, который можно использовать для шифрования.

X = —
Y = —
ВРЕМЯ = —
СУММА = X + Y + ВРЕМЯ = —
БИТ = СУММА % 2 = ?
Нажмите «Сгенерировать ключ»

Мастер-ключ успешно сгенерирован. Вы можете скопировать его для дальнейшего использования.
